# Product Knowledge Rules

- Use only metrics that appear in the files. Never invent or estimate numbers.
- Cite the file name when referencing a specific metric (e.g., "per company.md").
- When discussing competitors, reference competitive.md for threat levels and positioning.
- Cross-reference company.md for business snapshot (ARR, users, funding, team size).
