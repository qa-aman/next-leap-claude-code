# User Research Rules

- When synthesizing research, cite which interview or survey the finding comes from.
- Never invent user quotes. Only use quotes that appear in the transcript files.
- Connect findings to specific personas by name and link to their file in `05-user-personas/`.
- Look for patterns that appear across multiple sources - convergence is the signal.
