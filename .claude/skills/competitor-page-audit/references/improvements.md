# competitor-page-audit ‚Äî Team Feedback Log

Track feedback, bugs, and improvement ideas for the `competitor-page-audit` skill here.

## How to add feedback

1. Add a one-line row to the **Feedback queue** table (fast triage).
2. If your point needs detail (a repro, a proposed fix, example output), also add a **Detailed entry** block below using the copy-paste template.
3. Use **DD-MM-YYYY** for all dates.
4. Leave `Status` as `Open` when you add it. The skill owner moves it to `Triaged`, `In progress`, `Done`, or `Won't do`.

**Type** = `Bug` | `Enhancement` | `Copy/Content` | `Scoring` | `UX/Output` | `Praise`
**Priority** = `P1` (blocks a run) | `P2` (degrades output) | `P3` (nice to have)
