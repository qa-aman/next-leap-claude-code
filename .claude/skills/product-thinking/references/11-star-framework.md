# 11-Star Experience Framework

Brian Chesky (Airbnb) thought experiment: "What would a 1-star through 11-star experience look like?"

Forces teams to think beyond "good enough" and imagine transformative experiences.

## The Scale

| Star Level | Definition | Example Context |
|------------|-----------|----------------|
| **1-star** | Broken, unusable | Crashes, loses data, core function fails |
| **2-star** | Barely functional | Works but requires workarounds |
| **3-star** | Meets basic need | Functional but forgettable |
| **4-star** | Reliable, useful | Smooth, minor friction, gets the job done |
| **5-star** | Delights | Exceeds expectations, users recommend it |
| **6-star** | Memorable | Creates "wow" — something unexpected |
| **7-star** | Magical | Effortless, anticipates needs, proactive recommendations |
| **8-star** | Personalized magic | Adapts to individual patterns and preferences |
| **9-star** | Science fiction | Tech that doesn't exist yet |
| **10-star** | Impossible today | Industry-wide adoption |
| **11-star** | Transforms the domain | Reimagines what the domain IS |

## How to Use

### Step 1: Rate Your Feature

Describe what each star level looks like for THIS specific feature.

**Example — Data Export:**

| Star | Experience |
|------|-----------|
| 1-star | Export crashes; data corrupted |
| 3-star | CSV export with manual column selection; 30-second wait |
| 5-star | One-click export in multiple formats; instant |
| 7-star | Scheduled exports; auto-sends to stakeholders |
| 9-star | AI-generated insights sent proactively; no export needed |
| 11-star | Data transforms into decisions automatically |

### Step 2: Choose Target

- **Below 5-star:** Acceptable for MVP, communicate clearly
- **5-7 star:** Sweet spot for production
- **Above 7-star:** Aspirational; roadmap for future

### Step 3: Use as Scope Filter

- "4-star to 5-star?" — Worth it
- "5-star to 5.2-star?" — Probably not
- "Gets us to 7-star" — Realistic for current phase?

## Output Format

| Star Level | Experience Description | Realistic Now? |
|------------|----------------------|----------------|
| 1-star | [broken version] | — |
| 3-star | [basic version] | — |
| 5-star | [delightful version] | Yes/No |
| 7-star | [magical version] | Yes/No |
| 9-star | [sci-fi version] | No |
| 11-star | [transformative version] | No |

**Target Star Level:** {N} — {rationale}
