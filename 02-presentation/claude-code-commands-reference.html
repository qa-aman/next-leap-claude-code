Keyboard shortcuts

KEYS

The controls worth muscle-memorizing. macOS Alt/Option shortcuts need Option-as-Meta configured in your terminal.

EscInterrupt Claude mid-turn, or close an open dialog. Work done so far is kept.
Esc EscClear the input draft, or if empty, open the rewind menu to restore a previous state.
Ctrl+CInterrupt a running operation, or clear input. Press twice on empty input to exit.
Ctrl+DExit the Claude Code session (EOF).
Shift+TabCycle permission modes: Manual → acceptEdits → plan → any others you enabled.
Ctrl+OToggle the transcript viewer for detailed tool usage and execution.
Ctrl+RReverse-search your command history. Press again to cycle older matches.
Ctrl+TToggle Claude's to-do task checklist in the status area.
Ctrl+BMove a running Bash command or agent to the background. Tmux users press twice.
Ctrl+GOpen your prompt in your default external text editor.
Ctrl+LRedraw the screen if the display gets garbled. History is kept.
Ctrl+V / Cmd+VPaste an image from the clipboard as an [Image #N] chip.
Option+PSwitch model without clearing your prompt (Alt+P on Windows/Linux).
Option+TToggle extended thinking (Alt+T on Windows/Linux).
Option+OToggle fast mode (Alt+O on Windows/Linux).
Up / DownMove the cursor in multiline input; at the edges, navigate command history.
Shift+EnterMultiline input. Native in iTerm2, Ghostty, Kitty, Warp; run /terminal-setup for VS Code.
Ctrl+JInsert a newline in any terminal without configuration.
\ + EnterQuick multiline escape that works everywhere.
Tab / →Accept the grayed-out prompt suggestion.
