# body-heading, Figma spec

Text styles only, not a component. Apply per heading level.

## Text styles
- `text/h2` -> Be Vietnam Pro 36 / 44 / weight 600 / color `text-heading`
- `text/h3` -> 24 / 32 / weight 600 / color `text-heading`
- `text/h4` -> 20 / 28 / weight 600 / color `text-heading`
- responsive mobile variants: `text/h2-mobile` 28/36, `text/h3-mobile` 22/30

## Spacing rules
- h2: 64px top margin (`uxz-8`), 12px bottom (`uxz-3`)
- h3: 24px top (`uxz-5`), 12px bottom
- h4: 16px top (`uxz-4`), 8px bottom (`uxz-2`)

## Note
The source page does not use h2; the h2 style is interpolated for prototypes that need an extra hierarchy level.
