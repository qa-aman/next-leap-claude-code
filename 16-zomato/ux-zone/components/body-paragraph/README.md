# body-paragraph

Default article prose. Use the `.uxz-prose` wrapper once per article; every `<p>`, `<strong>`, `<em>`, `<a>` inside inherits styles.

- Source URL: https://www.zomato.com/blog/sushi/
- Source selector: `article p`, `article strong`, `article em`, `article a`
- Source quote (paragraph 1, verbatim): `Say hello to Sushi - Zomato's all-new design system that we've built from the ground up. It's not just a design system for us, but more than that it helps us provide a new and enhanced experience for our users using the Zomato app.`

## Tokens used
- typography: family-article, size-body, lh-body, weight-body
- color: text-body, text-heading, link
- spacing: space-4, space-5, space-6
