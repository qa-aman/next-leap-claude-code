# collection-card

Image-led card used in the "Collections" carousel. Image fills the card, gradient at the bottom carries title + place count.

- Source URL: https://www.zomato.com/ncr/restaurants
- Source selector: collection grid links, e.g. `a[href="/ncr/insta-worthy"]`
- Source quotes (verbatim): `Insta-worthy spots`, `82 Places`, `New in town`, `8 Places`, `Omakase bars`, `10 Places`

Place 6-10 cards inside a horizontal scroller for the full pattern. The kit ships one card; composition is up to the prototype.
