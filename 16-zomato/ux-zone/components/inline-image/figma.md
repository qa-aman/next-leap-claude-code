# inline-image, Figma spec

Frame: `inline-image` under `Components > Article`.

## Variants
None.

## Properties
| Property | Type | Default | Maps |
|---|---|---|---|
| `src` | Instance Swap | image fill |
| `caption` | Text | Atomic Design: Mapping with our design system | caption text layer |

## Auto-layout
- Direction: vertical
- Gap: 12px (`uxz-3`)
- Padding: 32px (`uxz-6`) top + bottom
- Align: stretch image to fill, center caption

## Text styles
- caption: `text/body-italic` (20 / 32.5 / weight 300 / italic / color text-body)
