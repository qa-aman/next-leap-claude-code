# tag-chip

Category label above an article title. On the source page this is plain uppercase text, no pill background. A `--brand` variant adds a red pill for usage outside article headers.

- Source URL: https://www.zomato.com/blog/sushi/
- Source selector: `article > div > div:first-child > div:first-child`
- Source quote: "technology"

## Variants
- default: uppercase, muted, no background
- brand: white text on Zomato red pill (extension, not on source page; use only when a stronger affordance is needed)
