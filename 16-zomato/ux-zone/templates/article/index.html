Zomato’s new Sushi Design System

Vijay Verma| 26-08-2019| 6 mins read

“The effort required to build a design system is tiny compared to the effort required to maintain it.” - Kyle Peatt

Say hello to Sushi – Zomato’s all-new design system that we’ve built from the ground up.

What is a Design System?

A design system is a large set of reusable components combined with a set of rules, storing visual design information, like colours or spacing, and enables us to build consistent applications quickly.

How did we come up with a brand new system?

Design Principle

Some core principles we kept in mind during this process were user centricity, inclusivity, simplicity, and consistency.

Creating Foundations

Foundations are digital brand guidelines, which define typography, color palettes, icons, spacing, shadow and information architecture of our design system.
